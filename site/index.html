Discord safety tool

PingGuard

A free Discord bot that detects image, GIF, video, sticker, and embed based mass-ping spam without downloading attachments or storing message content.

Add to Discord Privacy Terms

What it needs

  • Gateway intents: Guilds, Guild Messages, Message Content.
  • Permissions: View Channels, Read Message History, Manage Messages, Moderate Members, Send Messages, Embed Links.
  • No Administrator permission and no Guild Members Intent.
